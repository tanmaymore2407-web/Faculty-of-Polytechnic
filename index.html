Curriculum

What We Covered

Python skills feed the blockchain capstone. The blockchain capstone explains Polkadot. One flowing narrative across 40 days.

Python

Variables, control flow, functions, collections, file I/O, error handling

CS Fundamentals

Binary, computer architecture, data structures, algorithmic thinking

Internet & Networking

HTTP, DNS, TCP/IP — how the web actually works

Cloud Computing

Cloud models, GitHub Codespaces as a live cloud dev environment

Blockchain & Polkadot

SHA-256, block structures, PoW, PoS, relay chains, parachains, DOT, NPoS

Modules

Course Structure

Four modules plus special topics, each building on the last.

Module 1

Python Foundations

Core language: types, I/O, loops, functions, scope, debugging, refactoring. Capstones: Grade Buddy and GitHub Profile.

Weeks 1–3 · Days 1–15
Module 2

Python in Practice

Collections, string methods, file handling, exception handling, real-world program design.

Week 4 · Days 16–20
Module 3

Blockchain Basics

SHA-256 hashing, block construction, chain validation, PoW mining simulation, tamper detection. Capstone: Tiny Blockchain.

Weeks 5–6 · Days 21–30
Module 4

Polkadot & Web3

Relay chains, parachains, cross-chain messaging, DOT, NPoS. Live exploration of Subscan and Polkadot.js.

Weeks 7–8 · Days 31–40
Special Topics

Beyond the Syllabus

Computer Architecture · How the Internet Works · Cloud Computing — three deep-dives delivered between modules.
